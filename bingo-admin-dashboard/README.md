# BinGo Admin Dashboard

Simple web dashboard for viewing BinGo users.

## Setup

```bash
npm install
npm run dev
```

The dashboard expects the backend API at `http://localhost:5000` by default.
Set `VITE_API_BASE_URL` to use another backend URL.

```bash
VITE_API_BASE_URL=http://localhost:5000 npm run dev
```

## Build

```bash
npm run build
```
