# BinGo Tech Stack

## Backend
- **Framework:** Express (v5.x)
- **Language:** TypeScript
- **Database:** MongoDB / Mongoose (v9.x)
- **Authentication:** JWT, bcryptjs
- **AI Service:** @google/genai (Gemini 2.5-flash)

## Frontend
- **Framework:** React (v19)
- **Language:** TypeScript
- **Styling:** TailwindCSS
- **Build Tool:** Vite

## Mobile
- **Framework:** Flutter (3.x)
- **State Management:** Riverpod
- **Networking:** Dio
