## Current Focus
The primary goal is to address critical security, performance, and AI-service integration gaps identified during the project audit.

### Tracks
1.  **[Security & Authentication](./tracks/security-auth/index.md)**: Implement JWT-based auth, password hashing, and secure route protection.
2.  **[AI Service Refinement](./tracks/ai-service/index.md)**: Refactor Gemini service for structured outputs and async file I/O.
3.  **[Controllers & Routes Hardening](./tracks/controllers-routes/index.md)**: Secure endpoints, fix user identification, and ensure data integrity.
